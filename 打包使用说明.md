# Poster Again 打包使用说明

## 概述
本指南将帮助您将 Poster Again 程序打包为独立的 Windows exe 文件，用户无需安装 Python 环境即可使用。

## 准备工作

### 1. 安装依赖
```bash
# 安装 PyInstaller
pip install pyinstaller

# 确保已安装所有项目依赖
pip install -r config/requirements.txt
```

### 2. 确认图标文件
图标文件已自动生成：
- `resources/icon.ico` - Windows图标文件
- `resources/icon.png` - PNG预览

## 打包步骤

### 方法一：使用自动打包脚本（推荐）
```bash
# 运行打包脚本
python build_exe.py
```

打包完成后，exe文件将生成在 `dist/` 目录下。

### 方法二：手动执行 PyInstaller 命令
```bash
pyinstaller --onefile \
    --windowed \
    --name=PosterAgain \
    --icon=resources/icon.ico \
    --distpath=dist \
    --workpath=build \
    --specpath=. \
    --clean \
    --add-data=config/config.yaml;config \
    --add-data=config/*.pptcfg;config \
    --hidden-import=PySide6.QtCore \
    --hidden-import=PySide6.QtGui \
    --hidden-import=PySide6.QtWidgets \
    main.py
```

## 打包结果

打包完成后，`dist/` 目录结构：
```
dist/
├── PosterAgain.exe          # 主程序
├── config/                  # 配置文件目录
│   ├── config.yaml
│   ├── *.pptcfg
│   └── ...
└── examples/                # 示例文件目录
    ├── *.xlsx
    └── *.pptx
```

## 分发说明

### 1. 压缩包结构
将整个 `dist/` 目录打包为 zip 文件发送给用户，包含：
- **PosterAgain.exe** - 主程序（双击即可运行）
- **config/** - 配置文件目录（必须保持）
- **examples/** - 示例文件（可选）

### 2. 用户使用说明
用户解压后：
1. 双击 `PosterAgain.exe` 即可运行
2. 无需安装 Python 环境
3. 首次运行可能需要 Windows 防火墙允许

### 3. 注意事项
- **文件路径**：用户不应修改 `config/` 目录结构
- **权限**：确保 exe 文件有读取配置文件的权限
- **系统要求**：Windows 10/11，64位系统

## 故障排除

### 问题1：exe文件无法运行
**现象**：双击exe后没有反应或报错
**解决**：
1. 检查是否缺少依赖文件（config目录）
2. 以管理员身份运行
3. 查看 Windows 事件查看器中的错误信息

### 问题2：图标显示不正确
**现象**：exe图标显示默认图标
**解决**：
1. 确认 `resources/icon.ico` 文件存在
2. 删除临时文件后重新打包
3. 检查图标文件格式是否为有效 ICO

### 问题3：文件操作失败
**现象**：无法读取配置文件或保存文件
**解决**：
1. 检查 `config/` 目录是否完整
2. 确认用户有读写权限
3. 以管理员身份运行

### 问题4：exe文件过大
**说明**：PySide6等依赖库较大，30-50MB是正常大小
**优化**：
- 使用 `--exclude-module` 排除不需要的模块
- 压缩打包（UPX工具）

## 高级配置

### 自定义图标
如需自定义图标：
1. 设计 256x256 像素的 PNG 图片
2. 转换为 ICO 格式
3. 替换 `resources/icon.ico` 文件
4. 重新打包

### 减小文件大小
在 build_exe.py 中添加优化选项：
```python
--exclude-module=tkinter \
--exclude-module=matplotlib \
```

### 添加版本信息
创建 version.txt 文件并添加到打包命令中。

## 维护更新

如需更新程序：
1. 修改源代码
2. 重新运行打包脚本
3. 分发新的 exe 文件
4. 保留 config/ 目录（用户配置不会丢失）

## 常见问题 FAQ

**Q: exe文件运行时提示缺少 DLL？**
A: 确保 `--collect-all=PySide6` 参数已添加，或安装 Microsoft Visual C++ Redistributable

**Q: 如何设置 exe 的版本信息？**
A: 使用 PyInstaller 的 --version-file 参数，参考官方文档

**Q: 能否进一步压缩 exe 大小？**
A: 可以使用 UPX 工具压缩，但可能会影响启动速度

**Q: 如何处理用户数据目录？**
A: 程序应使用用户目录（如 %APPDATA%/PosterAgain）存储用户数据
