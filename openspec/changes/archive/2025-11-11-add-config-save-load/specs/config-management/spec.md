## ADDED Requirements

### Requirement: 配置文件保存功能
系统SHALL支持将当前的PPT占位符匹配规则和文本增加设置保存为配置文件。

#### Scenario: 用户成功保存配置
- **WHEN** 用户在GUI界面中点击"保存配置"按钮
- **AND** 当前已设置有效的占位符匹配规则
- **AND** 选择了保存路径和文件名
- **THEN** 系统创建包含所有匹配规则和文本增加设置的配置文件
- **AND** 配置文件包含版本信息、元数据和当前模板/数据的基本信息
- **AND** 保存成功后显示成功提示信息

#### Scenario: 用户尝试保存无效配置
- **WHEN** 用户在没有设置任何匹配规则时点击"保存配置"
- **THEN** 系统显示错误提示，说明没有有效的配置可保存
- **AND** 不创建配置文件

### Requirement: 配置文件加载功能
系统SHALL支持加载已保存的配置文件，并智能验证配置与当前模板和数据的兼容性。

#### Scenario: 配置文件完全兼容
- **WHEN** 用户选择加载配置文件
- **AND** 配置文件中的占位符与当前PPT模板完全匹配
- **AND** 配置文件中的数据字段与当前Excel文件完全匹配
- **THEN** 系统加载所有匹配规则和文本增加设置
- **AND** 更新GUI界面显示匹配结果
- **AND** 显示加载成功信息

#### Scenario: 配置文件部分兼容
- **WHEN** 用户选择加载配置文件
- **AND** 配置文件中的部分占位符或数据字段与当前文件不匹配
- **THEN** 系统仅加载兼容的配置项
- **AND** 跳过不兼容的配置项
- **AND** 向用户显示详细的兼容性报告，说明哪些配置被跳过

#### Scenario: 配置文件完全不兼容
- **WHEN** 用户选择加载配置文件
- **AND** 配置文件中的占位符或数据字段与当前文件完全不匹配
- **THEN** 系统拒绝加载配置
- **AND** 显示详细的兼容性错误信息
- **AND** 建议用户检查模板和数据文件

### Requirement: 配置文件格式和验证
系统SHALL使用标准化的配置文件格式，并包含完整的验证机制。

#### Scenario: 配置文件格式验证
- **WHEN** 系统尝试加载配置文件
- **THEN** 首先验证配置文件的JSON格式正确性
- **AND** 验证必需字段的存在性（version, config等）
- **AND** 验证配置文件版本的兼容性
- **AND** 如果格式验证失败，显示具体的错误信息

#### Scenario: 配置内容验证
- **WHEN** 配置文件格式验证通过后
- **THEN** 验证每个匹配规则的占位符和字段名称的有效性
- **AND** 验证文本增加规则的格式正确性
- **AND** 检查是否存在重复的占位符配置

### Requirement: 配置文件元数据管理
系统SHALL在配置文件中包含足够的元数据信息，便于用户识别和管理配置。

#### Scenario: 配置文件创建
- **WHEN** 用户保存配置时
- **THEN** 配置文件包含创建时间戳
- **AND** 包含当前使用的PPT模板文件名
- **AND** 包含当前使用的Excel数据文件名
- **AND** 包含PPT模板中的所有占位符列表
- **AND** 包含Excel数据中的所有字段列名

#### Scenario: 配置文件识别
- **WHEN** 用户浏览配置文件时
- **THEN** 系统能够显示配置文件的基本信息摘要
- **AND** 包含创建时间、模板信息、数据信息
- **AND** 显示配置中包含的匹配规则数量

## MODIFIED Requirements

### Requirement: ConfigManager扩展
现有的ConfigManager类SHALL扩展以支持占位符匹配配置的管理。

#### Scenario: 配置保存集成
- **WHEN** 调用ConfigManager的保存占位符配置方法
- **THEN** 将匹配规则和文本增加设置序列化为标准格式
- **AND** 调用现有的文件保存机制
- **AND** 记录配置保存操作日志

#### Scenario: 配置加载集成
- **WHEN** 调用ConfigManager的加载占位符配置方法
- **THEN** 验证配置文件格式和内容
- **AND** 将配置数据反序列化为内部数据结构
- **AND** 与现有的ExactMatcher集成
- **AND** 记录配置加载操作日志
