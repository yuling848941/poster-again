# PPT批量生成工具 - 使用指南（纯Python版本）

## 重要更新

**此版本使用纯 Python 的 python-pptx 库，无需安装 Microsoft Office 或 WPS！**

## 系统要求

- Windows 7/10/11
- Python 3.8+
- **无需 Microsoft Office**
- **无需 WPS Office**

## 安装步骤

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

主要依赖：
- `python-pptx` - PPT文件处理
- `pandas` - Excel数据读取
- `PySide6` - GUI界面
- `openpyxl` - Excel文件支持

### 2. 运行程序

```bash
python main.py
```

## 使用方法

### 第一步：准备模板文件

1. 创建或打开一个 `.pptx` 文件
2. 在需要替换内容的位置添加占位符

#### 占位符设置方法（推荐）

**方法1：使用 Alt Text（替代文本）⭐推荐**

1. 在 PowerPoint 中右键点击形状
2. 选择"编辑替代文本"（Edit Alt Text）
3. 输入占位符名称，格式为：`ph_字段名`
   - 例如：`ph_姓名`、`ph_公司名称`、`ph_金额`

**方法2：使用形状名称**

1. 在 PowerPoint 中打开"选择窗格"（Home → Select → Selection Pane）
2. 将形状名称改为占位符格式：`ph_字段名`
   - 例如：`ph_姓名`、`ph_公司名称`

**方法3：使用文本标记**

直接在文本框中输入：
```
姓名：{{姓名}}
公司名称：{{公司名称}}
```

### 第二步：准备数据文件

创建 Excel 文件（`.xlsx`），包含与占位符对应的列：

| 姓名 | 公司名称 | 金额 |
|------|----------|------|
| 张三 | ABC公司 | 10000 |
| 李四 | XYZ公司 | 20000 |

**列名匹配规则：**
- 精确匹配：`ph_姓名` 匹配 `姓名` 列
- 忽略大小写：`ph_姓名` 匹配 `姓名` 列
- 忽略空格：`ph_营销员姓名` 匹配 `营销员 姓名` 列
- 忽略特殊字符：`ph_家族_DESC` 匹配 `家族 DESC` 列

### 第三步：使用GUI生成PPT

1. **选择模板文件**：点击"浏览..."选择 `.pptx` 模板
2. **选择数据文件**：点击"浏览..."选择 `.xlsx` 数据文件
3. **选择输出目录**：点击"浏览..."选择保存位置
4. **自动匹配**：点击"自动匹配"按钮，系统会自动匹配占位符和数据列
5. **调整匹配**（可选）：在匹配表格中手动调整对应关系
6. **批量生成**：点击"批量生成"按钮生成PPT文件

## 占位符命名建议

| 用途 | 示例 |
|------|------|
| 人员姓名 | `ph_姓名`、`ph_营销员姓名` |
| 公司/组织 | `ph_公司名称`、`ph_支公司` |
| 金额数字 | `ph_金额`、`ph_标保`、`ph_保费` |
| 日期时间 | `ph_日期`、`ph_递交时间` |
| 产品信息 | `ph_险种`、`ph_产品名称` |
| 地区信息 | `ph_城市`、`ph_省份` |

## 注意事项

### 1. 格式保留
- python-pptx 会尽可能保留原始格式
- 字体、颜色、大小等样式会被保留
- 动画效果会被保留在 `.pptx` 文件中

### 2. 特殊功能限制
- **导出图片**：python-pptx 不支持直接导出为图片，如需此功能请使用 LibreOffice
- **复杂动画**：某些复杂动画可能需要 Office 软件才能完全呈现

### 3. 性能优化
- 大量数据（>1000行）生成时可能需要较长时间
- 建议分批处理大量数据

## 常见问题

### Q: 程序提示找不到占位符？
A: 请检查：
1. 占位符是否以 `ph_` 开头
2. Alt Text 是否正确设置（右键形状 → 编辑替代文本）
3. 数据文件的列名是否与占位符匹配

### Q: 生成的PPT格式错乱？
A: 请检查：
1. 模板文件是否为标准 `.pptx` 格式
2. 替换的文本长度是否过长（可能导致文本框溢出）
3. 字体在系统中是否可用

### Q: 如何设置多个占位符？
A: 一个形状可以包含多个文本标记：
```
恭喜 {{姓名}} 来自 {{公司名称}}
达成 {{金额}} 业绩！
```

### Q: 是否支持图片替换？
A: 当前版本主要支持文本替换。图片替换功能需要额外开发。
