# PPT生成器

一个基于PySide6和Python的PPT批量生成工具，支持从Excel、CSV和JSON文件读取数据，并使用PowerPoint模板自动生成个性化PPT文档。

## 项目概述
这是一个基于Pyside6开发的自动化工具，可以根据Excel数据批量生成个性化的PPT。该项目的核心功能是保持PPT的复杂特效的同时，实现批量ppt生成，适用于制作工牌、证书、海报等场景。

## 功能特点

- 📊 **多数据源支持**：支持Excel、CSV和JSON格式的数据文件
- 🎯 **精确匹配**：精准匹配数据字段与模板占位符，确保数据准确填充
- 🖼️ **GUI界面**：提供直观的图形用户界面，操作简单易用
- ⌨️ **命令行支持**：支持命令行操作，方便集成到自动化流程
- ⚙️ **灵活配置**：提供丰富的配置选项，满足不同使用场景
- 📝 **详细日志**：记录操作过程，便于问题排查

## 核心功能
1. PPT模板处理 ：
   
   - 支持保持复杂的PPT特效（3D、辉光、映像、阴影等）
   - 使用占位符命名约定（ph_字段名）自动映射数据
   - 支持文本占位符的动态替换
2. 多数据源支持 ：
   
   - Excel (.xlsx)
   - CSV (.csv)
   - JSON (.json)
3. 双模式运行 ：
   
   - GUI模式 ：提供友好的图形界面，支持文件选择、参数配置、进度监控
   - CLI模式 ：命令行模式，支持自动化脚本和批处理

## 使用方法

### GUI界面

1. 启动应用程序：
   ```
   python main.py
   ```

2. 选择模板文件：
   - 点击"浏览"按钮选择PowerPoint模板文件
   - 模板中应包含占位符，格式为`ph_字段名`，在PowerPoint的"选择窗格"中设置元素名称

3. 选择数据文件：
   - 点击"浏览"按钮选择数据文件（Excel、CSV或JSON）
   - 系统会自动识别文件格式

4. 设置输出目录：
   - 选择生成PPT文件的保存位置

5. 配置选项：
   - 设置文件名前缀和后缀
   - 调整匹配阈值
   - 其他高级选项

6. 生成PPT：
   - 点击"生成"按钮开始批量生成
   - 查看进度和结果

### 命令行界面

基本用法：
```
python cli.py --template 模板文件.pptx --data 数据文件.xlsx --output 输出目录
```

高级选项：
```
python cli.py \
  --template 模板文件.pptx \
  --data 数据文件.xlsx \
  --output 输出目录 \
  --config 配置文件.json \
  --prefix 文件名前缀 \
  --suffix 文件名后缀 \
  --threshold 0.8 \
  --preview
```

参数说明：
- `--template`: PowerPoint模板文件路径
- `--data`: 数据文件路径（Excel、CSV或JSON）
- `--output`: 输出目录路径
- `--config`: 配置文件路径（可选）
- `--prefix`: 生成文件名前缀（可选）
- `--suffix`: 生成文件名后缀（可选）
- `--threshold`: 匹配阈值（0-1，可选）
- `--preview`: 预览模式，只显示匹配结果不生成文件（可选）

## 模板制作指南

1. 创建PowerPoint模板文件
2. 在需要填充数据的位置添加占位符，格式为`ph_字段名`
3. 占位符应该设置在PowerPoint的"选择窗格"中的元素命名，而不是直接在文本中
4. 保存模板文件

示例：
- 姓名元素命名为：`ph_姓名`
- 部门元素命名为：`ph_部门`
- 日期元素命名为：`ph_日期`

## 数据文件格式

### Excel格式
- 支持.xlsx和.xls格式
- 数据从第一行开始，第一行为列标题
- 每行数据对应一个生成的PPT文件

### CSV格式
- 支持UTF-8编码
- 第一行为列标题
- 使用逗号分隔字段

### JSON格式
- JSON数组格式
- 每个对象代表一行数据
- 对象的键为字段名，值为字段值

示例：
```json
[
  {
    "姓名": "张三",
    "部门": "技术部",
    "职位": "工程师"
  },
  {
    "姓名": "李四",
    "部门": "市场部",
    "职位": "经理"
  }
]
```

## 配置文件

应用程序使用JSON格式的配置文件，默认配置文件为`config.json`。配置文件包含以下部分：

- `app`: 应用程序基本配置
- `ppt`: PowerPoint相关配置
- `data`: 数据读取配置
- `matching`: 字段匹配配置
- `logging`: 日志记录配置

## 常见问题

### Q: 为什么无法连接到PowerPoint？
A: 请确保已安装Microsoft PowerPoint，并且有足够的权限启动应用程序。

### Q: 数据匹配不准确怎么办？
A: 可以调整匹配阈值，或检查数据字段名和模板占位符是否一致。

### Q: 生成的PPT文件打不开？
A: 请确保模板文件没有损坏，并且有足够的磁盘空间。

### Q: 如何批量处理大量数据？
A: 建议将数据分批处理，每批不超过1000条记录，以避免内存不足。
