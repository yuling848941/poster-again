# 保单号可复制功能修改说明

## 修改目的

**用户需求**：在"承保趸期数据批量输入"面板中，将保单号从"灰色不可选中"改为"只读但可复制"，方便用户复制保单号进行查询。

## 修改前 vs 修改后

### 修改前
- **显示**：QLabel 显示保单号
- **状态**：完全不可选中
- **操作**：无法复制
- **视觉效果**：

```
行号: 第 1 行          保单号: 123456789          缴费年期: [输入框]
```

### 修改后
- **显示**：QLineEdit (只读模式) 显示保单号
- **状态**：只读但可选中
- **操作**：支持 Ctrl+C 复制
- **视觉效果**：

```
行号: 第 1 行    保单号: [123456789]    缴费年期: [输入框]
```

## 技术实现

### 修改文件
`src/gui/chengbao_term_input_dialog.py`

### 关键修改 (第131-155行)

**替换前**：
```python
# 保单号标签
policy_label = QLabel(f"保单号: {policy_number}")
policy_label.setMinimumWidth(150)
policy_label.setStyleSheet("color: #6c757d;")
row_layout.addWidget(policy_label)
```

**替换后**：
```python
# ✅ 新增：保单号显示框（只读但可复制）
policy_field = QLineEdit(str(policy_number))
policy_field.setReadOnly(True)  # 设置为只读，不可更改
policy_field.setMinimumWidth(150)
policy_field.setMaximumWidth(200)

# 设置样式：看起来像不可编辑但可以选中和复制
policy_field.setStyleSheet("""
    QLineEdit {
        color: #6c757d;
        background-color: #f8f9fa;
        border: 1px solid #dee2e6;
        border-radius: 3px;
        padding: 4px 8px;
        selection-background-color: #007bff;
    }
    QLineEdit:focus {
        border-color: #007bff;
    }
""")

# ✅ 新增：提示文本，告诉用户可以复制
policy_field.setToolTip("可复制此保单号进行查询")

row_layout.addWidget(policy_field)
```

## 使用场景

### 场景1：复制单个保单号
1. 在批量输入对话框中
2. 点击保单号文本框
3. 选中保单号（可选中部分或全部）
4. 按 Ctrl+C 复制
5. 切换到查询系统
6. 按 Ctrl+V 粘贴查询

### 场景2：批量记录保单号
1. 在对话框中逐行记录保单号
2. 选中某个保单号
3. 复制并粘贴到记事本或其他记录工具
4. 方便后续批量查询
