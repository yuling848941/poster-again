# PPT占位符配置保存加载功能使用指南

## 功能概述

本功能允许用户保存和加载PPT占位符与Excel数据字段的匹配配置，包括文本增加规则（前缀/后缀），避免重复配置工作。

## 功能特点

- ✅ **配置保存**: 将当前占位符匹配规则保存为 `.pptcfg` 文件
- ✅ **智能加载**: 自动检查配置与当前模板/数据的兼容性
- ✅ **部分兼容**: 只加载兼容的配置项，跳过不兼容项
- ✅ **详细提示**: 显示兼容性检查结果和加载详情
- ✅ **配置管理**: GUI界面友好的配置文件管理

## 如何使用

### 1. 启动应用程序

```bash
python main.py
```

### 2. 设置占位符匹配

1. 选择PPT模板文件
2. 选择Excel数据文件
3. 点击"自动匹配"按钮或手动设置匹配规则
4. 可选择性设置文本增加规则（前缀/后缀）

### 3. 保存配置

1. 点击底部的绿色 **"配置管理"** 按钮
2. 在弹出的对话框中点击 **"保存当前配置"**
3. 选择保存路径和文件名（建议使用 `.pptcfg` 扩展名）
4. 点击保存

**配置文件包含内容**:
- PPT占位符与Excel字段的对应关系
- 文本增加规则（前缀/后缀设置）
- 模板和数据文件信息
- 版本和元数据

### 4. 加载配置

1. 点击底部的绿色 **"配置管理"** 按钮
2. 在弹出的对话框中点击 **"加载配置文件"**
3. 选择之前保存的 `.pptcfg` 文件
4. 系统会自动进行兼容性检查

### 5. 兼容性检查

系统会检查：
- **占位符兼容性**: 当前PPT模板的占位符与配置文件的匹配度
- **数据字段兼容性**: 当前Excel文件的列名与配置文件的匹配度

**兼容性结果**:
- 🟢 **完全兼容**: 直接加载所有配置
- 🟡 **部分兼容**: 显示兼容性检查结果，用户可选择加载兼容的规则
- 🔴 **不兼容**: 提示用户检查模板和数据文件

## 配置文件格式

配置文件是标准的JSON格式，示例：

```json
{
  "version": "1.0.0",
  "created_at": "2025-11-11T08:27:30.936677",
  "metadata": {
    "template_info": {
      "file_name": "template.pptx",
      "placeholders": ["ph_name", "ph_title", "ph_company"]
    },
    "data_info": {
      "file_name": "data.xlsx",
      "columns": ["姓名", "职位", "部门", "薪资"]
    },
    "rule_count": 2
  },
  "matching_rules": [
    {
      "placeholder": "ph_name",
      "column": "姓名",
      "text_addition": {
        "prefix": "",
        "suffix": "先生"
      }
    },
    {
      "placeholder": "ph_title",
      "column": "职位",
      "text_addition": {
        "prefix": "职位：",
        "suffix": ""
      }
    }
  ]
}
```

## 常见问题

### Q: 什么时候需要保存配置？
A: 当您设置了复杂的占位符匹配关系，或者经常使用相同的模板和数据结构时，建议保存配置以便下次快速加载。

### Q: 配置文件保存在哪里？
A: 您可以选择任意位置保存，建议保存在项目文件夹或专门的工作目录中。

### Q: 配置文件可以分享吗？
A: 可以！配置文件是标准的JSON格式，可以分享给其他使用相同模板的用户。

### Q: 如果换了模板或数据文件怎么办？
A: 加载配置时系统会自动检查兼容性，只会应用兼容的配置项。

### Q: 如何判断配置是否兼容？
A: 系统会显示详细的兼容性报告，包括：
- 整体兼容率（百分比）
- 具体哪些规则兼容
- 具体哪些规则不兼容及原因

## 最佳实践

1. **命名规范**: 使用有意义的文件名，如 `销售报告配置.pptcfg`
2. **定期备份**: 将重要的配置文件备份到安全位置
3. **版本管理**: 对于重要配置，可以保存不同版本
4. **团队协作**: 与团队成员共享配置文件，确保一致性

## 技术说明

### 文件扩展名
- 配置文件使用 `.pptcfg` 扩展名
- 实际上是标准的JSON格式文本文件

### 版本兼容性
- 当前支持版本 1.0.0
- 未来版本会保持向后兼容

### 数据安全
- 配置文件不包含实际的PPT或Excel数据
- 只保存匹配规则和元数据信息

## 故障排除

### 保存失败
- 检查是否有有效的匹配规则
- 确认保存路径有写入权限

### 加载失败
- 检查文件是否损坏
- 确认文件格式是否为有效的JSON

### 兼容性问题
- 确认占位符命名一致（以 `ph_` 开头）
- 检查Excel列名是否匹配
- 查看详细的兼容性报告
