# 递交趸期数据功能说明

## 功能概述

在GUI应用的"自定义"按钮菜单中新增了"递交趸期数据"选项，目前仅作为占位符功能，尚未实现具体业务逻辑。

## 使用方法

1. 在匹配结果表格中，点击任意行的"自定义"按钮
2. 在弹出的下拉菜单中，选择"递交趸期数据"选项
3. 系统将显示提示信息，表明该功能待实现

## 当前实现

目前该功能仅包含以下实现：

1. **菜单项添加**：在`show_custom_menu`方法中添加了"递交趸期数据"菜单项
2. **事件处理**：创建了`submit_term_data`方法，目前仅显示提示信息
3. **用户反馈**：通过日志和消息框提示用户该功能待实现

## 代码位置

主要修改的文件：
- `src/gui/main_window.py`：添加了菜单项和事件处理方法

## 后续开发

待实现的具体功能包括：
1. 获取当前行的占位符信息
2. 设计趸期数据递交的界面
3. 实现数据递交的业务逻辑
4. 添加递交结果的处理和反馈

## 测试

已创建测试脚本 `test_submit_term_data.py` 验证菜单项的添加和基本功能调用。

## 注意事项

1. 目前该功能仅为占位符，不执行任何实际操作
2. 点击该选项不会影响现有功能
3. 后续实现时需考虑与现有数据流的集成
