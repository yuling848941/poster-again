# PPT批量生成工具使用说明

## 概述

PPT批量生成工具是一个基于Python开发的图形界面应用程序，可以根据Excel数据批量生成PPT文件。该工具支持自动匹配PPT模板中的占位符与数据表中的列，实现快速批量生成个性化PPT。

## 功能特点

- **图形界面操作**：直观易用的图形界面，无需命令行操作
- **多格式数据支持**：支持Excel、CSV、JSON等数据格式
- **智能占位符匹配**：自动匹配PPT模板中的占位符与数据表中的列
- **实时进度显示**：显示处理进度和详细日志信息
- **灵活配置**：支持自定义匹配规则和输出选项

## 安装与启动

### 环境要求

- Python 3.8 或更高版本
- Windows 操作系统

### 安装步骤

1. 克隆或下载项目代码
2. 创建虚拟环境（推荐）：
   ```
   python -m venv .venv
   .venv\Scripts\activate
   ```
3. 安装依赖包：
   ```
   pip install -r requirements.txt
   ```

### 启动应用程序

运行以下命令启动GUI应用程序：
```
python main.py
```

## 使用步骤

### 1. 选择模板文件

1. 点击"模板文件"输入框旁的"浏览..."按钮
2. 选择包含占位符的PPT模板文件（.pptx格式）
3. 占位符格式为"ph_字段名"，例如"ph_姓名"、"ph_部门"等

### 2. 选择数据源文件

1. 点击"数据源文件"输入框旁的"浏览..."按钮
2. 选择包含数据的Excel、CSV或JSON文件
3. 数据文件应包含与模板占位符对应的列

### 3. 选择输出目录

1. 点击"输出目录"输入框旁的"浏览..."按钮
2. 选择生成的PPT文件保存位置

### 4. 自动匹配占位符

1. 选择模板文件和数据文件后，点击"自动匹配"按钮
2. 系统会自动尝试匹配模板中的占位符与数据文件中的列名
3. 匹配规则：
   - 精确匹配：占位符名与列名完全相同
   - 部分匹配：占位符名包含列名或列名包含占位符名
4. 匹配结果会显示在匹配表格中，绿色表示已匹配，红色表示未匹配
5. 如果有未匹配的占位符，可以手动编辑匹配规则

### 批量生成PPT

1. 确认匹配关系正确后，点击"批量生成"按钮
2. 系统会开始批量生成PPT文件
3. 可以通过进度条和日志信息查看处理进度
4. 文件名格式可以在生成前设置，支持使用{index}作为索引占位符
5. 生成完成后会显示成功生成的文件数量

### 注意事项
- 确保输出目录有足够的存储空间
- 生成大量PPT文件可能需要较长时间，请耐心等待
- 如果数据中有空值，对应的占位符将不会被替换

## 占位符格式说明

PPT模板中的占位符应使用以下格式：
- 前缀："ph_"
- 示例：`ph_姓名`、`ph_部门`、`ph_职位`

## 注意事项

1. 确保PPT模板文件中的占位符格式正确
2. 数据文件中的列名应与占位符相对应
3. 生成过程中请勿关闭应用程序
4. 输出目录需要有足够的磁盘空间

## 常见问题

### 1. 自动匹配功能报错
- **问题**: 点击自动匹配按钮时提示"'PPTGenerator' object has no attribute 'get_data_columns'"
- **解决**: 确保使用的是最新版本的应用程序，该问题已在v1.1版本中修复

### 2. 占位符未匹配
- **问题**: 自动匹配后仍有占位符显示为红色（未匹配）
- **解决**: 
  - 检查数据文件中是否有对应的列
  - 手动在匹配表格中选择正确的列名
  - 确保占位符格式正确（以"ph_"开头）

### 3. PPT生成失败
- **问题**: 批量生成时部分PPT文件生成失败
- **解决**:
  - 检查数据文件中是否有空值
  - 确保输出目录有写入权限
  - 检查模板文件是否损坏

### 4. 应用程序启动失败
- **问题**: 启动应用程序时出现错误
- **解决**:
  - 确保已安装所有依赖项：`pip install -r requirements.txt`
  - 检查Python版本是否为3.8或更高
  - 确保config.yaml文件存在且格式正确

### 5. 批量生成失败
- **问题**: 点击批量生成按钮后出现错误或生成失败
- **解决**:
  1. 确保所有占位符都已正确匹配
  2. 检查输出目录是否有写入权限
  3. 确保数据中没有空值或格式错误
  4. 检查模板文件是否完整且可正常打开
  5. 尝试只生成少量数据进行测试

### Q: 支持哪些数据格式？
A: 目前支持Excel（.xlsx）、CSV（.csv）和JSON（.json）格式的数据文件。

### Q: 生成的PPT文件命名规则是什么？
A: 默认命名规则为"output_序号.pptx"，可以在配置文件中修改命名模式。
