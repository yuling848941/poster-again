# 文本增加功能使用说明

## 功能概述

文本增加功能允许用户在生成PPT时为占位符添加前缀和后缀，从而实现更灵活的文本格式化。例如，可以为姓名添加"尊敬的"前缀和"先生"后缀，为职位添加"（高级）"后缀等。

## 使用方法

### 1. 通过GUI界面使用

1. 启动GUI应用程序：`python main.py`
2. 选择模板文件和数据源文件
3. 点击"自动匹配"按钮进行占位符匹配
4. 在匹配结果表格中，点击需要添加文本的行右侧的"自定义"按钮
5. 在弹出的菜单中选择"增加文本"
6. 在对话框中输入前缀和后缀，点击"确定"
7. 表格中的数据列将显示为"[自定义] 前缀:xxx 后缀:xxx"
8. 点击"批量生成"按钮生成PPT

### 2. 通过代码使用

```python
from src.ppt_generator import PPTGenerator

# 创建PPT生成器实例
generator = PPTGenerator()

# 加载数据
generator.load_data("data.xlsx")

# 设置匹配规则
matching_rules = {
    'ph_name': 'name',
    'ph_title': 'title',
    'ph_department': 'department'
}
generator.exact_matcher.set_matching_rules(matching_rules)

# 设置文本增加规则
generator.set_text_addition_rule('ph_name', prefix='尊敬的', suffix='先生')
generator.set_text_addition_rule('ph_title', suffix='（高级）')
generator.set_text_addition_rule('ph_department', prefix='所属部门：')

# 获取应用文本增加规则后的数据
row_data = generator.exact_matcher.get_all_data_for_row(0)
print(row_data)  # {'ph_name': '尊敬的张三先生', 'ph_title': '工程师（高级）', 'ph_department': '所属部门：技术部'}

# 批量生成PPT
generator.batch_generate("template.pptx", "output/", matching_rules)
```

## API参考

### PPTGenerator类

#### 文本增加规则相关方法

- `set_text_addition_rule(placeholder, prefix="", suffix="")`: 为指定占位符设置文本增加规则
- `get_text_addition_rule(placeholder)`: 获取指定占位符的文本增加规则
- `get_all_text_addition_rules()`: 获取所有文本增加规则
- `remove_text_addition_rule(placeholder)`: 移除指定占位符的文本增加规则

### ExactMatcher类

#### 文本增加规则相关方法

- `set_text_addition_rule(placeholder, prefix="", suffix="")`: 为指定占位符设置文本增加规则
- `get_text_addition_rule(placeholder)`: 获取指定占位符的文本增加规则
- `get_all_text_addition_rules()`: 获取所有文本增加规则
- `remove_text_addition_rule(placeholder)`: 移除指定占位符的文本增加规则
- `apply_text_addition(value, prefix="", suffix="")`: 应用文本增加规则到值
- `get_all_data_for_row(row_index)`: 获取指定行的所有数据，并应用文本增加规则

## 示例

假设有以下数据：

| name | title | department |
|------|-------|------------|
| 张三 | 工程师 | 技术部 |
| 李四 | 设计师 | 设计部 |

设置以下文本增加规则：

- `ph_name`: 前缀="尊敬的", 后缀="先生"
- `ph_title`: 后缀="（高级）"
- `ph_department`: 前缀="所属部门："

生成的PPT中的文本将变为：

- 姓名: 尊敬的张三先生
- 职位: 工程师（高级）
- 部门: 所属部门：技术部

## 注意事项

1. 文本增加规则只影响最终生成的PPT，不会修改原始数据
2. 如果同时设置了前缀和后缀，它们会分别添加到原始值的前后
3. 可以只设置前缀或只设置后缀，另一个留空即可
4. 文本增加规则会保存在匹配规则文件中，下次加载时会自动应用
