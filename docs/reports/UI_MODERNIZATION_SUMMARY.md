## 设计特点

### 配色方案
- **主色调**: 品牌蓝 #2563EB
- **强调色**: 渐变绿 #10B981 → #059669
- **语义色**:
  - 成功: #10B981
  - 警告: #F59E0B
  - 错误: #EF4444
- **中性色**: 完整的灰度系统 (#F9FAFB → #111827)

### 动画效果
- 按钮悬停: 颜色加深 + 阴影增强 (300ms)
- 按钮按下: 缩放0.98倍 (150ms)
- 焦点过渡: 平滑过渡 (200ms)

### 字体层级
- 标题: 16px, Black weight
- 正文: 14px, Normal weight
- 标签: 12px, Medium weight
- 按钮: 14px, Medium/Bold weight
