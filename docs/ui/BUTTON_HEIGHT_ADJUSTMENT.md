# 按钮高度调整完成

## 调整内容

根据用户反馈，将所有按钮高度从 35px 调整为 **18px**（原来的一半），使界面更加紧凑协调。

### 修改的文件

#### `src/gui/ui_constants.py`

**1. 高度常量调整**
```python
# 修改前
HEIGHT_BUTTON = 35
HEIGHT_COMBOBOX = 35

# 修改后  
HEIGHT_BUTTON = 18
HEIGHT_COMBOBOX = 18
```

**2. 按钮内边距优化**
```python
# 修改前
PADDING_BUTTON = 12

# 修改后
PADDING_BUTTON = 4
```

## 样式机制

按钮高度通过以下方式统一管理：

1. **CSS 样式表**: `min-height: {Dimensions.HEIGHT_BUTTON}px`
2. **代码设置**: `setMinimumHeight(Dimensions.HEIGHT_BUTTON)`
3. **常量统一**: 所有按钮使用同一个常量值
