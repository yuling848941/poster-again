## 🔧 自定义选项

### 修改程序名称
编辑 `build_exe.py` 第65行：
```python
--name=YourAppName,
```

### 自定义图标
1. 替换 `resources/icon.ico`
2. 重新运行打包脚本

### 添加额外文件
在 `build_exe.py` 中添加：
```python
--add-data=your_file;your_folder,
```
